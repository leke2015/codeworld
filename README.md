# 我是一个码农 - codeworld
## 介绍

【码农的世界】的小程序，本来想给小程序起个和公众号一样的名字。结果腾讯不让。本来想给公众号起个和网站一样的名字 -【码农的世界】，结果这个名字注册了。所以...

所以嘛，网站叫【码农的世界】，公众号叫【码农的微世界】，小程序叫【我是一个码农】。

第一次做小程序，没啥经验。直接 git clone 了一个。稍微改了改。就用了。除了能在上面回评论。也没啥交互性功能，有点鸡肋。

主要就是想了解下微信小程序开发的是怎样的一个过程。
